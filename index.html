Configuring Git

  1. Go to VS code and click Terminal, arrow then terminal
  2. Check if git is already installed by type git --version in the command line
  3. Configure git in your local computer:
    • Set your Username
      git Config --global user.name "your username"
    • Set your email address
      git config --global user.email "email@sample"
  4. To initialize the project directory, type git init

Git add and commit

  1. To see the status of the file in your directory, type git status
  2. To track new files, use the command git add ;
    Syntax git add . for group files
    Syntax git add file name for individua files.
  3. By running the git status again, you can see the staged files in green ready for commit.
  4. Commit the files that you have staged in your local repository by typing git commit -m
    Sample: Syntax git commit -m "message"

Git Branches

Git Branch is a new version of the main repository.
It allows you to work on different parts of the project w/out impacting the main branch.
  1. To list Branches: Syntax git branch
  2. To create new branch: Syntax git branch (new branch name)
  3. To switch on new branch: Syntax git checkout (branch name)
